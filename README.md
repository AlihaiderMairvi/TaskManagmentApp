A simple and clean task management application built with Flutter that allows users to manage their daily tasks efficiently. The app supports adding, deleting, and marking tasks as complete with local data persistence.
Add Tasks - Easily add new tasks using the + icon in the app bar
Delete Tasks - Remove unwanted tasks with the delete icon
Mark Complete - Tap on any task to toggle between complete/incomplete status
Visual Indicators - Green check icon for completed tasks, grey circle for pending tasks
Data Persistence - All tasks are automatically saved using SharedPreferences
Splash Screen - Professional loading screen when the app starts
